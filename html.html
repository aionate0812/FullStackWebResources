Tips and tricks

  • Beautifier Site | Beautifier VS extension

    There are websites and Visual Studio extensions that allow you to format your code by addding indentations to your code and make it more readable. If you are using a website just copy and paste your code in the input textbox click the button to format it and copy and paste the formated code back into your editor.

    If you are using Visual Studio you can download the Beautify extension here. Once you install it just press fn+f1 on Mac a search bar will appear, type '> Beautify [file] or [selection]'. If you are using beautify selection make sure to select the text first

  • HTML Validator

    HTML is a languange and as any languange it has its own grammar, vocabulary and syntax. A validator verifies that the code or file follows the rules
